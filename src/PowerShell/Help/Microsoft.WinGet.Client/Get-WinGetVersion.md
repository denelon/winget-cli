# Get-WinGetVersion

## SYNOPSIS
Get the installed version of WinGet.

## SYNTAX

```
Get-WinGetVersion [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Get the installed version of WinGet.

## EXAMPLES

### Example 1: Default example
```powershell
PS C:\> Get-WinGetVersion
```

Get the installed version of WinGet.
